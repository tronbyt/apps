# Custom Animation

Display a single custom image with full support for animated GIFs.

## Features

- **Custom Animation Display**: Upload and display your own custom animated GIF or static image
- **Animated GIF Support**: Full support for animated GIFs
- **Static Image Support**: PNG, JPG, and BMP formats
- **Automatic Animation**: Animated GIFs play automatically and loop continuously

## Supported Formats

- **GIF** - Static and animated (recommended for animations)
- **PNG** - Static images
- **JPG/JPEG** - Static images
- **BMP** - Static images

## Configuration

### Image
Upload a single custom image using the PhotoSelect field. Animated GIFs are recommended for the best animation experience.

## Usage Notes

- Animated GIFs will play their full animation cycle and loop continuously
- The animation speed is determined by the GIF file itself, not the app
- Images are automatically scaled to 64x32 pixels to fit the Tidbyt display
- **Important**: Use small file sizes (under 100KB recommended) to avoid configuration save failures
- If no image is selected, a placeholder message will be displayed
- Images are stored as base64-encoded data in your configuration
