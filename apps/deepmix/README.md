# Deep Mix - Tidbyt App

A Tidbyt app that displays the currently playing song from Deep Mix Online Radio in real-time.

## Preview

![Deep Mix App Preview](app.webp)

*The app displays the currently playing track with customizable colors and smooth scrolling text.*

## About

This app connects to Deep Mix Online Radio's live stream API to show what's currently playing on your Tidbyt display.

### Configuration

The app supports the following configuration options:

- **Text Color**: Color for track information text.
