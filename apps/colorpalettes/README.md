# Color Palettes for Tronbyt
Displays a random user submitted palette from [Colourpod](https://www.colourpod.com/). The user can also choose to show palettes that have certain colors.

![Color Palettes for Tronbyt](colorpalettes.webp)<br>
*when you walk through a wall but there's no wall at all, that's a doorway*, by [fastman27](https://www.colourpod.com/post/804103694028242944/)

## About the palettes
There are 7,284 palettes in `palettes.star`, and each one contains:
- `id`: the palette's post ID that correspondes to `https://www.colourpod.com/post/ID`
- `codes`: hex codes for the palette, formatted as a list of strings with each code in the format `#a1b2c3`
- `colors`: colors of the palette, formatted as a list of strings containing one or more of `red`, `orange`, `yellow`, `green`, `blue`, `purple`, `pink`, or `brown`

Palettes were last harvested from Colourpod on: **January 20, 2026**
