# Split Flap Display

A high-fidelity simulation of a mechanical split-flap display (like a Vestaboard) for Tronbyt.

## Features
- **3-Line Grid**: A 3x7 grid that fills the screen.
- **Mechanical Animation**: Authentic shuffle animation with randomized timing.
- **Color Flaps**: Use special symbols to display solid color flaps.
- **Centering**: Optional auto-centering for text.
- **Random Colors**: Optional random color background for empty spaces.

## Color Codes
Use these symbols in any of the 3 text lines to display a solid color flap:

| Symbol | Color |
| :--- | :--- |
| `[` | Red |
| `]` | Orange |
| `{` | Yellow |
| `}` | Green |
| `(` | Blue |
| `)` | Indigo |
| `<` | Violet |
| `>` | White |

## Configuration
- **Line 1-3**: The text to display on each row (max 7 characters).
- **Random Color Spaces**: Toggle to replace all empty spaces with random color flaps.
- **Center Text**: Toggle to center the words on each line.
